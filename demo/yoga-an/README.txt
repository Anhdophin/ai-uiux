Linh An Yoga - static HTML demo

Structure
- assets/styles: design tokens, base, layout, reusable component styles
- assets/images: provided demo images from assets.zip
- content: structured content modules per page
- features: section renderers and local form behavior
- pages: route-level composition only
- shared: site shell and generic render helpers

Notes
- The visual pack from assets.zip is used as a placeholder demo pack, so a few images are intentionally treated as decorative rather than literal yoga photography.
- Contact form is demo-only and validates client-side without any backend connection.
