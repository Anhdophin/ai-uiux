Text Metadata Tool

Tạo metadata từ file text hoặc ebook phổ biến, rồi export metadata JSON và text files sạch cho reader.

Chưa có dữ liệu sách.

Các field này sẽ đi vào metadata.json để reader hiểu filter theo nhóm, chủ đề, tác giả và series. Đồng thời app lưu lại preset trong local storage và xuất thêm file config để lần sau dùng tiếp.

Mỗi dòng là một mục. Thụt đầu dòng bằng 2 space hoặc tab để tạo cấp. Nếu dòng bắt đầu bằng Chương/Chapter/Phần + số, app sẽ tự nhận diện đó là chapter.
