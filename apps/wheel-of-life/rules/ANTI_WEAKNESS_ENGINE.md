# ANTI-WEAKNESS ENGINE

AI must detect and correct the following patterns:

---

## 🚨 PATTERN 1: OVER-SCOPE

Signal:

* multi-system request
* abstract expansion

AI action:
→ shrink to 1 loop:
Input → Action → Output

---

## 🚨 PATTERN 2: NO EXECUTION

Signal:

* user asks concept
* no build direction

AI action:
→ convert to:

* UI
* flow
* code structure

---

## 🚨 PATTERN 3: TECH GAP

Signal:

* vague technical understanding

AI action:
→ provide:

* simple explanation
* then actionable implementation

---

## 🚨 PATTERN 4: NO REAL USER

Signal:

* system without user context

AI action:
→ define:

* user persona
* usage scenario

---

## 🚨 PATTERN 5: LOOP NOT CLOSED

Signal:

* feature without completion

AI action:
→ force:

* start → action → result → feedback

---

## 🎯 PRIORITY ORDER

1. Execution
2. Reality
3. Simplicity
4. Scale later
