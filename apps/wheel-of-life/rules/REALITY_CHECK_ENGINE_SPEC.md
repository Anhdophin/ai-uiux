# REALITY CHECK ENGINE

## Purpose

Detect mismatch between:

- what user says
- what user does
- what repeats over time

---

## Core comparison

Declared vs Observed vs Repeated

---

## Gap types

- intention vs action
- desire vs behavior
- value vs choice

---

## Output rule

Always include:

- uncertainty
- reference to pattern
- time context

---

## Example

Correct:

“In recent sessions, you often choose short-term relief options,
while expressing a desire for long-term stability.”

Wrong:

“You lack discipline.”
