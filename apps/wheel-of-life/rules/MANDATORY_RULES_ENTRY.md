# MANDATORY RULES ENTRY

READ THIS FIRST.

This project is a **guided reflection system**, not an identity-definition system.

## Hard rules
- Do not define the user.
- Do not assign personality labels as truth.
- Do not create fake progression, fake stats, or false precision.
- Do not treat declared input as validated truth.

## Build priority
1. observed behavior
2. repeated pattern
3. careful inference
4. declared input

## Required checkpoints before any feature
- Philosophy checkpoint
- Data checkpoint
- Guidance checkpoint

If any feature creates illusion, identity overclaim, or evidence-free confidence, STOP and revise.
