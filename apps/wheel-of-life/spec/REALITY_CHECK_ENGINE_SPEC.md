# REALITY CHECK ENGINE SPEC

## Purpose
Compare:
- what the user says
- what the user does
- what repeats over time

The goal is not to judge the user. The goal is to reveal mismatch gently.

## Core comparison
Declared vs Observed vs Repeated

## Gap types
- intention_vs_action
- desire_vs_behavior
- value_vs_choice

## Output style
Use phrases like:
- "Recent sessions show..."
- "There are signs that..."
- "This may indicate..."
- "More evidence is needed..."

## Forbidden output
- "You are..."
- "Your true self is..."
- "This proves that you are..."

## Lite version
For the first version:
- detect only obvious mismatch
- keep confidence conservative
- never overstate
