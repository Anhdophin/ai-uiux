# REFLECTION ENGINE FOUNDATION

## Purpose
Build a structured reflection environment where users interact mainly through guided choices.

## North star
> Do not help users believe a fantasy self.  
> Help them see what they keep repeating.

## Core loop
Context -> AI suggests -> User chooses -> System logs -> Pattern emerges -> AI reflects

## Design principles
- selection-first
- mobile-first
- icon-first
- minimal typing
- auditable output
- no free-chat dependency

## What counts as truth here
Not identity claims.
Not self-description alone.
Truth emerges from:
- repeated choices
- repeated avoidance
- repeated return behavior
- contradiction across time
