# REFLECTION SYSTEM BUNDLE README

This bundle is organized so both humans and AI agents can re-understand the project quickly.

## Folder roles

### /rules
These files define the doctrine and hard constraints.
Read these first before coding or planning.

### /spec
These files explain how the system should work.
They are architectural references for implementation.

### /prompt
These files are reusable prompt templates for AI-assisted building or runtime logic.

### /examples
These files provide simple example data for testing understanding and alignment.
