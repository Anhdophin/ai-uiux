# Prompt Strategy

## Goal
Biến context rời rạc thành context snapshot gọn, rõ, có thể kiểm soát.

## Prompt sections
1. Persona summary
2. Feeling summary
3. Selected keyword trail
4. Reflection snippets
5. Wheel preview
6. Current AI mode
7. Safety reminder

## Rule
Prompt text và assembly phải nằm ở backend, không nằm trong UI.
