# LONG SESSION RESUME STRATEGY

Users may leave and return later. Resume should restore context clearly.

## Resume inputs
- last selected feeling
- selected keyword IDs
- wheel preview
- latest prompt mode
- recent trail window
- last compaction summary if present

## Resume outputs
- short session summary
- next suggested action
- optional prompt mode to continue with
- evidence-backed reason for that suggestion

## Design principle
Resume should help the user continue by choosing, not by reading a long transcript.
